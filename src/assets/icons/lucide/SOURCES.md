# Lucide icon sources

All SVG files in this directory are local copies of icons from the official
[Lucide repository](https://github.com/lucide-icons/lucide/tree/main/icons).

Icons added during the final UI audit on 2026-07-16 were downloaded from:

`https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/{icon-name}.svg`

The applicable Lucide ISC license and Feather-derived MIT notice are preserved
in `LICENSE.txt` in this directory.
